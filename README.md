# Basic interpreter in Rust

## Format 

### Operations on integers

```
1 + 1
1 - 1
1 * 1
1 / 1
-2
```

### Group operations

```
(1 + 2) * 2
```

### Python-like Assignment

```
a = 1;
(1 + a) * 2
```
